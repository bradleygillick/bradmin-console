# brAdministrator-console

The brAdministrator Console is a Resource Monitor for the local machine.

# Deployment
Clone this repo.
CD in to project directory.
`npm install`
`npm start dev`

## Built with

* Angular
* Express
* Node.js
* Angual Chart
