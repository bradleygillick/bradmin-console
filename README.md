# brAdministrator-console

The brAdministrator Console is a Resource Monitor for your local Mac.

# Deployment
* Clone this repo.
* CD in to project directory.
* Run `npm install` in the terminal
* Run `npm start dev` in the terminal

## Built with

* Angular
* Express
* Node.js
* Angual Chart
